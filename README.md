# 🐍 Python Data Management System

Aplicación desarrollada en Python con arquitectura modular, enfocada en la gestión de datos, validación de entradas y organización del código en componentes reutilizables.

---

## 🚀 Funcionalidades

- Gestión de datos estructurada
- Validación de entradas del usuario
- Separación de lógica por módulos
- Flujo controlado desde archivo principal
- Código limpio y mantenible
